# Smart Building

Веб-приложение для жильцов и управляющей компании.

## Возможности
- авторизация
- платежи
- счётчики
- заявки
- голосования
- объявления
- чат
- профиль
- административная панель

## Тестовые аккаунты
Жилец1: resident@house.kz / 12345
Администратор: admin@house.kz / admin
Жилец2:apt23@house.kz / pass:'12345
